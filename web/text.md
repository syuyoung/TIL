# text-decoration

| 값 | 의미 |
| --- | --- |
| none | 선 없음 |
| underline | 밑줄을 지정 |
| overline | 윗줄을 지정 |
| line-through | 중앙 선(가로지르는)을 지정 |

# letter-spacing

문자의 자간(글자 사이 간격)을 설정

| 값 | 의미 | 기본값 |
| --- | --- | --- |
| normal | 단어 사이의 일반 간격 | normal |
| 단위 | px, em, cm 등 단위로 지정 |   |

# word-spacing

단어 사이(띄어쓰기)의 간격을 설정

| 값 | 의미 | 기본값 |
| --- | --- | --- |
| normal | 단어 사이의 일반 간격 | normal |
| 단위 | px, em, cm 등 단위로 지정 |   |
